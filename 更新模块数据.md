### 1. 更新任务提交
接口信息

- 请求路径 : PUT /api/case-task-submissions
- 请求方法 : PUT
- 接口描述 : 更新任务提交的基本信息（标题、内容、类型）
请求参数

参数名 类型 必填 说明 id Long 是 提交ID submissionTitle String 否 提交标题 submissionContent String 否 提交内容 submissionType String 否 提交类型

请求示例

```
{
  "id": 1,
  "submissionTitle": "更新后的标题",
  "submissionContent": "更新后的内容
  ",
  "submissionType": "NORMAL"
}
```
响应示例

```
{
  "code": 200,
  "message": "success",
  "data": {
    "id": 1,
    "caseTaskId": 1,
    "submissionTitle": "更新后的标题
    ",
    "submissionContent": "更新后的内
    容",
    "submissionType": "NORMAL",
    "submissionNumber": 1,
    "status": "APPROVED",
    "creatorName": "张三",
    "fileCount": 2,
    "createTime": 
    "2026-01-26T10:00:00",
    "updateTime": 
    "2026-01-26T12:00:00"
  }
}
```
### 2. 更新任务提交文件描述
接口信息

- 请求路径 : PUT /api/case-task-submissions/{submissionId}/files/{fileId}
- 请求方法 : PUT
- 接口描述 : 更新任务提交文件的描述信息
请求参数

参数名 类型 必填 说明 submissionId Long 是 提交ID（路径参数） fileId Long 是 文件ID（路径参数） description String 否 文件描述（查询参数）

请求示例

```
PUT /api/case-task-submissions/1/
files/123?description=更新后的文件描述
```
响应示例

```
{
  "code": 200,
  "message": "success",
  "data": null
}
```
### 3. 批量更新任务提交
接口信息

- 请求路径 : PUT /api/case-task-submissions/batch
- 请求方法 : PUT
- 接口描述 : 批量更新任务提交，支持同时更新提交信息和文件操作
请求参数

参数名 类型 必填 说明 id Long 是 提交ID submissionTitle String 否 提交标题 submissionContent String 否 提交内容 submissionType String 否 提交类型 fileOperations Array 否 文件操作列表

fileOperations 参数说明

参数名 类型 必填 说明 fileId Long 是 文件ID operation String 是 操作类型：DELETE-删除文件，UPDATE-更新文件描述 description String 否 文件描述（当operation为UPDATE时需要）

请求示例

```
{
  "id": 1,
  "submissionTitle": "批量更新后的标题
  ",
  "submissionContent": "批量更新后的
  内容",
  "submissionType": "NORMAL",
  "fileOperations": [
    {
      "fileId": 123,
      "operation": "UPDATE",
      "description": "更新后的文件描述
      "
    },
    {
      "fileId": 124,
      "operation": "DELETE"
    }
  ]
}
```
响应示例

```
{
  "code": 200,
  "message": "success",
  "data": {
    "id": 1,
    "caseTaskId": 1,
    "submissionTitle": "批量更新后的
    标题",
    "submissionContent": "批量更新后
    的内容",
    "submissionType": "NORMAL",
    "submissionNumber": 1,
    "status": "APPROVED",
    "creatorName": "张三",
    "fileCount": 1,
    "createTime": 
    "2026-01-26T10:00:00",
    "updateTime": 
    "2026-01-26T12:00:00"
  }
}
```
